# Windows Tests

This folder holds the Vitest coverage for the Windows-native installer MVP.

## Coverage goals

- Phase 1 bootstrap coherence
- environment check behavior
- shell state transitions
- phase-one orchestration
- diagnostics helper behavior

## Commands

From the `Windows/` directory:

- `npm test` — run the test suite once
- `npm run test:watch` — run Vitest in watch mode
- `npm run typecheck` — check the Windows package types
